Welcome to the SiMSAM Information Page!

SiMSAM stands for Simulation, Measurement, and Stop Action Moviemaking. It combines the playful, expressive, easy-to-use features of SAM Animation with the precision and testability of programming environments like NetLogo and StageCast Creator.

Can I try SiMSAM out?

We will be releasing a public version of SiMSAM in June, and will provide a link here then. If you'd like to be notified when it is available, please join our mailing list. SiMSAM runs best on Firefox and Chrome.

How do I install SiMSAM?

SiMSAM runs entirely in your browser, so you don't need to install anything to use it!

If you want to set up a new instance of SiMSAM that would run on your own computer or a different server, here is some information: SiMSAM runs on the django web framework. We have used it successfully with both sqlite3 and mySQL databases. If you are familiar with these, then it should be relatively easy to get the SiMSAM code available here downloaded and up and running on your own system. However, as is the case with many such applications, you may need to change some of the settings that come with the download. We can't guarantee we will be able to help you get set up, but will try to be helpful within reason.

Why did you build SiMSAM?

We are researchers interested in building upon and connecting the familiar, everyday ways young people make sense of their world to powerful ideas in mathematics, science, and computation. We built SiMSAM to see if by more closely connecting drawing and animation to computer programming and measurement, students would feel more ownership over the programs they create.

We have been using SiMSAM in workshops and classrooms with middle school students to explore everyday scientific processes like diffusion, evaporation, and sound propagation. We do research on how students think and learn with tools like SiMSAM, and how we can better support them.

To learn more about us, visit the Expressive Technologies Lab.
